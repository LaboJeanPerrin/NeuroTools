The folder 'Compute' contains functions used to compute different things

driftCompute 		computes the drift using cross correlation in Fourier space
mapToReferenceBrain	uses imregdemons to find displacement matrix to fit the given reference brain
caToolsRunquantile	uses the R module 'caTools' to compute quickly the running quantile
dff			computes signal - baseline / baseline - background for each pixel along in the ROI
